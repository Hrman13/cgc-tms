# CGC Transport Management System

A web-based Transport Management System developed for CGC University.

## Features
- Student Login
- Driver Login
- Admin Dashboard
- Bus Route Management
- Student Transport Details
- Driver Information
- Chat Section
- Transport Incharge Details

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Project Status
Currently under development. New features and improvements are being added.

## Disclaimer
This is an academic project created for learning and demonstration purposes only. It is not an official CGC University application.
